# GenAIus landing page

A single-page pitch built for Felix Schmidt at eMerge Americas 2026, booth SS23. The design mirrors genai.us directly: black field, IBM Plex Sans as the sole face, the 73-degree aurora gradient used sparingly (hero rule, CTA wash, halo, framework accent), and the short-punchy-to-long-action cadence Schmidt uses in his own copy. The page carries the full pitch body from the GenAIus fit analysis section 7, the 60/30/10 frame applied to an Executive AI Suite, the Feeld precedent as the matching case study, the ICM paper as the matching methodology, and a Calendly CTA against Matt Creamer for a peer-to-peer partnership conversation. No pricing is surfaced. All URLs are real anchors opening in a new tab. Eduba branding lives only in the footer and in named case references. Mobile responsive to 375px and below.
